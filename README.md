## Overview

The Math Quiz Game is an educational Python program designed to make learning and practicing math fun and engaging for users of all ages. Math is a fundamental skill, and this game aims to help users improve their math abilities through an enjoyable and interactive experience.

The game generates random math problems, consisting of addition and multiplication operations, and asks users to solve them. It provides immediate feedback on the correctness of the answers and keeps track of users' scores. This README.md file provides comprehensive information about the project, its features, how to use it, and how to contribute to its development.

## Features

### 1. Random Problem Generation

The Math Quiz Game generates math problems with varying complexity to challenge users and provide a wide range of learning opportunities. Problems include random integers between 0 and 100 and use either addition or multiplication operations.

### 2. Interactive User Interface

The game offers an easy-to-use, text-based interface with clear instructions. Users can quickly understand how to play and enjoy the learning experience without any confusion.

### 3. Score Tracking

The program keeps track of the user's score, allowing them to monitor their progress and strive for improvement. Positive feedback with emojis encourages users when they answer correctly.

### 4. Replayability

Users can play the game multiple times to continue practicing and honing their math skills. The option to exit gracefully allows users to finish their session whenever they choose.

## Getting Started

### Prerequisites

Before running the Math Quiz Game, ensure you have the following prerequisites installed on your system:

- Python 3.x

### Installation

1. Clone the Repository:

   ```shell
   git clone https://github.com/KemoMedhat/math-quiz-game.git
   cd math-quiz-game
   ```

2. Run the Game:

   ```shell
   python math_quiz_game.py
   ```

## Usage

1. Launch the game by running the `math_quiz_game.py` script.

2. Follow the on-screen instructions to play the game:

   - You will be presented with a math problem to solve.
   - Enter your answer.
   - Receive immediate feedback on whether your answer is correct or not.
   - Choose to play again or exit.

The Math Quiz Game provides a simple yet effective way to enhance your math skills while having fun.
